Leveraging AI in the Local Government

December 2024

Intensify the data driven culture in the Richmond and Wandsworth Council. Image by OpenAI.



C-Suite Challenges:

  1. Require insights from data-led tools to balance resource allocation and optimize staffing efficiency.
  2. Aim to optimise the process of getting intelligence but reducing the workload of the current data scientist team at the same time.
  3. Seek an AI-driven approach to streamline data analysis and reduce operational costs for call center management.


My Solutions/Chatbot Overview:

  1. Data Integration: The AI chatbot, powered by AI, seamlessly integrates NetCall data from multiple sources (Azure Databricks, SQL, etc; liteSQL and CSV in the demo) into a unified platform for real-time analysis and reporting.
  2. Natural Language Querying: Empowers users to ask questions in plain English and receive instant insights through AI-generated SQL queries or Pandas-based analysis, removing the need for technical expertise in querying.
  3. Actionable Insights: Provides real-time analytics, such as average time (days) to close a request per call request category, enables data-driven decision-making to optimize operations.
  4. Cost Optimization: Executes queries without resorting to agentic actions.
  5. Transparent results: queries and insights are all visible and transparent. Avoids the black-box processes in agentic architecture.
  6. Scalable and Secure: Built on a scalable architecture leveraging Azure’s cloud ecosystem, ensuring secure, high-performance access to data and insights.
